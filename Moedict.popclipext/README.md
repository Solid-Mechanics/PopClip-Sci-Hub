# Moedict

A PopClip Extension to search selected text on [Moedict](https://www.moedict.tw/).

## Text

The selected text should be within the range of [CJK Unified Ideographs](https://en.wikipedia.org/wiki/CJK_Unified_Ideographs) (4E00â€“9FFF) with a maximal length of 20 characters.
