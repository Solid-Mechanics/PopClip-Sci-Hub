# WP Shortcode Converter

A PopClip Extension to convert fenced code block to Wordpress Shortcode format.

## Examples

    ```bash
    chmod a+x everwatch.rb
    ```

will be converted to:

```
[bash]
chmod a+x everwatch.rb
[/bash]
```
